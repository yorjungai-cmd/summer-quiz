# 🌟 Summer Quiz — v1.2.4

Web application สำหรับติวสอบน้องซัมเมอร์ ชั้น ป.1 โรงเรียนอัสสัมชัญสมุทรปราการ  
สร้างด้วย Next.js 15 · Deploy บน Vercel · รองรับ iPad, Tablet, มือถือทุกรุ่น  
🔗 [summer-quiz-mu.vercel.app](https://summer-quiz-mu.vercel.app)

---

## เกี่ยวกับโปรเจกต์

แอปติวสอบที่ออกแบบให้น้อง ป.1 ใช้บน iPad หรือมือถือได้ง่าย ไม่ต้องติดตั้งแอปเพิ่มเติม  
ระบบหลักเป็นแบบ RPG battle — ตอบถูกโจมตีมอนสเตอร์ ตอบผิดเสีย HP — เพื่อสร้างแรงจูงใจในการเรียน  
มีระบบ Mini Games เสริมสำหรับฝึกความจำในรูปแบบที่หลากหลาย

## Features

### Quiz System
- ข้อสอบแบบ 4 ตัวเลือก เลือกจำนวนข้อได้ (10 / 20 / 50 / 100) สุ่มจากคลังข้อสอบทุกครั้ง
- ระบบ RPG battle: Hero vs Monster พร้อม HP bar, animation โจมตี/เสีย HP
- ต้องกด "ยืนยัน" ก่อนถึงรู้ผล — ไม่มี feedback ระหว่างเลือก
- หลังจบมีหน้า Results สรุปคะแนน พร้อม Reward ตามเปอร์เซ็นต์
- Review Mode — ดูเฉพาะข้อที่ตอบผิด พร้อมคำอธิบายทั้งเฉลยและเหตุผลที่ตัวเลือกอื่นผิด
- ระบบ EXP / Level สะสมข้ามเซสชัน บันทึกลง localStorage

### Mini Games
| เกมส์ | รายละเอียด |
|-------|-----------|
| 🎴 จับคู่ภาพ-คำ | การ์ด 16 ใบ 4x4 grid พอดีหน้าจอทุกอุปกรณ์ — emoji ใหญ่ด้านหนึ่ง คำ+emoji ประกอบอีกด้านหนึ่ง |
| ✏️ เติมคำในประโยค | 12 ข้อ เลือกคำที่หายไปจากประโยค มีคำใบ้กำกับทุกข้อ |
| 🔤 เรียงคำเป็นประโยค | 8 ข้อ กดเรียงคำให้ถูกลำดับ แสดงเฉลยเมื่อตอบผิด |

### เนื้อหาภาษาไทย (100 ข้อ)
- เพื่อนรู้ใจ — reading comprehension เนื้อเรื่องภูผาและช้างน้อย
- ช้างน้อยน่ารัก — reading comprehension การแสดงช้างประจำหมู่บ้าน
- สะกดคำและความหมาย — คำศัพท์จากบทเรียน
- ปริศนาคำทาย — ปริศนาเกี่ยวกับพืชและผลไม้
- อักษรสูง / อักษรต่ำ — การผันเสียงวรรณยุกต์
- มาตราตัวสะกด — แม่กง กน กม กย กว กบ
- สัญลักษณ์และป้ายจราจร

### Compatibility
- รองรับ iOS Safari (iPad/iPhone), Android Chrome, Desktop Browser
- Responsive ด้วย `clamp()` และ `min()` ปรับขนาดอัตโนมัติตามหน้าจอ
- การ์ดเกมส์จับคู่ใช้ `min(calc((100vmin - 130px) / 4), 155px)` — พอดีจอทุกขนาดโดยไม่ต้อง scroll
- ปุ่มหลัก `position:fixed` ที่ bottom — กดได้เสมอไม่ว่าจอจะสูงแค่ไหน
