# 🌟 มาช่วยติวน้องซัมเมอร์สอบกันเถอะ!! — v1.1.0

## 📁 โครงสร้างโปรเจกต์
```
summer-quiz/
├── app/
│   ├── layout.tsx
│   └── page.tsx
├── components/
│   ├── SummerQuiz.jsx      ← แอปหลัก (ระบบ Quiz)
│   └── MiniGames.jsx       ← เกมส์เสริมความจำ 3 เกมส์
├── questions/
│   ├── Thai.js             ← ภาษาไทย 100 ข้อ ✅
│   ├── Math.js             ← คณิตศาสตร์ 🔒 รออัพเดท
│   └── MiniGameData.js     ← ข้อมูลเกมส์ (จับคู่, เติมคำ, เรียงคำ)
└── package.json
```

## 🎮 เกมส์เสริมความจำ
- **🎴 จับคู่ภาพ-คำ** — พลิกการ์ดจับคู่ emoji กับคำศัพท์
- **✏️ เติมคำในประโยค** — เลือกคำที่ขาดหายจากประโยค
- **🔤 เรียงคำเป็นประโยค** — กดเรียงคำให้ถูกลำดับ
